**📚 StudySpark – AI‑Powered Smart Study Planner**
StudySpark is a full‑stack intelligent study management system designed to help students plan tasks, prepare for exams, track productivity, log Pomodoro sessions, and receive personalized AI‑generated study tips.
It features IST‑accurate scheduling, auto‑generated timetables, weak‑subject detection, chatbot assistance, and a modern dashboard for complete study workflow automation.

✨ Features
🕒 Indian Standard Time (IST) Engine
Displays live IST clock across dashboard, schedule, and Pomodoro timer.

All study sessions, logs, and schedules are stored using Asia/Kolkata timezone.

Automatically skips past time slots and starts from the next available 15‑minute window.

📅 Auto‑Generated Smart Timetable
Calculates daily availability windows (e.g., 6 PM – 9 PM).

Fits tasks into free slots using a greedy bin‑packing algorithm.

Generates Critical Revision tasks for exams approaching in <10 days.

Allocates more time to weak subjects automatically.
