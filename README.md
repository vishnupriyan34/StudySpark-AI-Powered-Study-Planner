# 📚 **StudySpark — AI-Powered Study Planner** 
### 🔥*Spark your plan... Light up your grades...*

**🚀StudySpark is a full‑stack, AI‑powered web application built to help students plan, organize, and stay on top of their exam preparation. It generates personalized daily/weekly study plans based on subjects, exam dates, tasks, and available study time — all synced to Indian Standard Time (IST).**
----------------------------------------------
**🚀 Live Project**
👉 https://studysparkedu.onrender.com/
----------------------------------------------
📖 About the Project

Students often struggle with managing tasks, study time, exam prep, and revision schedules. StudySpark solves this by combining a smart rule-based AI engine with an intuitive planner interface — helping students study with focus instead of stress.
Built as a complete end-to-end system with authentication, scheduling algorithms, a Pomodoro focus timer, an AI chatbot assistant, and an admin dashboard.


🚀 Features

🧠 AI-Powered Scheduling — Rule-based engine generates personalized study plans using priority, due dates, and subject difficulty
🎯 Weak Subject Detection — Automatically flags weak subjects and allocates 1.5x more study sessions to them
⏰ Live IST Clock — Real-time day, date, and time tracking (Asia/Kolkata timezone)
🍅 Pomodoro Focus Timer — Built-in timer with session logging to track focused study time
💬 AI Chatbot Assistant — Ask "What should I study today?" and get instant, accurate answers
📊 Smart Study Tips — Dynamic tips based on subject difficulty, exam proximity, and session length
🌗 Light/Dark Mode — Clean, modern UI with theme toggle and education-themed illustrations
🗂️ Admin Dashboard — Manage students, monitor activity, and track platform metrics
📅 Calendar Export — Export generated schedules as .ics files (Google Calendar/Outlook compatible)
🖨️ Print-Friendly Timetables — Export and print study plans directly from the browser

🛠️ Tech Stack
Frontend
⚛️ React (Vite)
🎨 Vanilla CSS (custom design system, light/dark themes)

Backend
🟢 Node.js + Express
🔐 JWT Authentication
🗃️ JSON File-Based Database (zero-config, fully portable, no native binaries)
Core Logic:
📐 Custom scheduling algorithm (priority + due date + difficulty based)
🤖 Rule-based AI engine (offline, no external API keys required)

⚙️ Core Algorithm Highlights:
🕒 All scheduling is handled in IST (Asia/Kolkata), skipping time slots that have already passed today
📌 Tasks are ordered by priority → due date proximity → subject difficulty
📈 Weak subjects automatically receive extra study sessions
💡 Each session includes a contextual tip — practice-based, revision-based, or understanding-based — depending on subject and urgency
🧩 Use Cases
🎓 Exam Prep — Auto‑generate timetables based on exam dates.
📅 Daily Planning — Organize tasks into IST‑accurate schedules.
📊 Productivity Tracking — Monitor focus hours and Pomodoro logs.

💡 AI Study Tips — Get personalized revision and learning advice.
💬 Chatbot Help — Ask for study guidance and motivation.

👨‍💼 Admin Monitoring — Track student performance and publish updates.

🎓 Student Dashboard — Manage tasks, view timetables, and chat with AI.
